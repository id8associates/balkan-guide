# Balkan Guide — Marketing Plan

## Overview
Balkan Guide (balkanguide.net) is a multi-country tourism portal covering all 10 Balkan countries with 190 destinations, restaurants, accommodations, and things to do. Monetization is primarily through Booking.com affiliate commissions, display ads, and future digital products.

---

## 1. SEO Strategy (Highest Priority — Free Traffic)

### On-Page SEO
- Each destination page already has unique descriptions, meta titles, and structured data
- Add **schema.org/Article** markup for each destination (JSON-LD)
- Add **breadcrumb schema** for navigation
- Add **FAQ schema** on country pages

### Technical SEO
- Site is already on Cloudflare CDN (fast loading)
- Create an XML sitemap that includes all country and destination URLs
- Submit sitemap to Google Search Console + Bing Webmaster Tools
- Ensure all 190 images have alt text matching destination names

### Content Strategy
- Blog section: "Best Time to Visit the Balkans", "Balkan Travel Itineraries", "Hidden Gems"
- Each blog post targets long-tail keywords like "best beaches in Albania" or "how to get from Dubrovnik to Kotor"
- Target 2-3 new articles per week

### Keyword Targets
- "Balkan travel guide" (high volume)
- "Best places to visit in [Country]" (medium volume)
- "[Destination] travel tips" (low competition)
- "Balkan itinerary 14 days" (medium competition)
